# DoorKnock - Interactive Door Sounds
DoorKnock adds knock sounds to doors in Minecraft, enhancing immersion and player interaction. Players can knock on doors with different variations, while server administrators have full control over permissions and configuration.

## Features
- Sounds - Different knock sounds for various door materials
- Soft Knocks - Hold Shift for quiet knocking
- Permission System - Fine-grained control over who can knock and hear
- Color Support - Full hex color support in messages
- Customizable - Extensive configuration options

## How to Use
- Normal Knock: Left-click on any door
- Soft Knock: Hold Shift + Left-click for quiet knocking
- Admin Commands: ``/doorknock reload`` to refresh configuration
  
## Commands & Permissions
### Commands
- ``/doorknock reload`` - Reload plugin configuration
- ``/dk reload`` - Short alias for reload command
### Permissions
- ``doorknock.knock`` - Allows players to knock on doors (default: true)
- ``doorknock.hear`` - Allows players to hear door knocks (default: true)
- ``doorknock.admin`` - Allows admin commands like reload (default: op)
